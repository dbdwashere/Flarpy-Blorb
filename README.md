# 🐤 Flappy Bird Clone in Unity

A simple yet complete Flappy Bird clone built with **Unity** and **C#**.

✅ Basic 2D mechanics  
✅ Gravity and physics  
✅ Score tracking  
✅ Pipe spawning system  
✅ Game over and restart features
